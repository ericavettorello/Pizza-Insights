**Pizza Insights** is a demo project that showcases how to integrate **MCP servers, a REST API, and a Telegram bot** in a single solution.  
The project runs inside **Cursor**, enabling order data analysis (CSV ‚Üí DuckDB), generating reports via **API (FastAPI)**, which is accessible as a website in any browser, and displaying key metrics through a **Telegram bot**.
